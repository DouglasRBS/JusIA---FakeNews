# ⚖️ JusIA - Justiça Inteligente contra a Desinformação

A **JusIA** é uma plataforma inteligente desenvolvida para detectar fake news e combater a desinformação de forma acessível, inclusiva e simples.

O projeto foi criado com foco em acessibilidade, especialmente para pessoas com deficiência ou com pouca familiaridade digital.

## 📌 Objetivo

A JusIA tem como objetivo:

- Combater fake news
- Analisar textos, imagens e vídeos
- Classificar conteúdos como:
  - ✅ Verdadeiro
  - ⚠️ Duvidoso
  - ❌ Falso
- Tornar a verificação de informações acessível para todos

## 🚀 Funcionalidades

- 🔎 Verificação de textos, manchetes e links
- 🖼️ Análise simulada de imagens e vídeos
- 📢 Leitura em voz alta para acessibilidade
- 🌐 Simulação de integração com redes sociais
- 📱 Interface simples e intuitiva
- 🎨 Design focado em usabilidade

---

## 🧠 Como Funciona

1. O usuário insere um texto, link ou manchete.
2. Pode anexar imagem ou vídeo.
3. O sistema realiza uma análise simulada.
4. O conteúdo é classificado como:
   - Confiável
   - Duvidoso
   - Falso
5. O usuário recebe um retorno visual simples e direto.
